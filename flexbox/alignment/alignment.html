Alignment

Let's use this example.

What happens if we add flex: 1 to .item?

The three items are stretched evenly within the flex container to fill up the available width.

What if we wanted to have each container retain it's width and distribute the available space evenly? Then we remove flex: 1 and add justify-content: space-between to .container.



justify-content is used to align items along the main axis. This property is applied to the item's container. Now apply the value center to justify-content. This brings the items back the way it was originally but the items are brought to the center of the container.

Now what if we wanted to align our items along the cross axis? We have to add a new property called align-items. To align our items to the center of the container, we now add align-items: center.

Recall from our last lesson in Axes, the main axis and cross axis will change depending on our flex container's flex-direction. By default, our flex-direction is set to row. So justify-content will align the main axis (orizontal) and align-items will align items vertically. This is the default if no declarations were made to flex-direction.

When flex-direction is set to column, then justify-content aligns vertically and align-items aligns horizontally.

Gap

The property gap on the flex container adds a specified gap between flex items. It is a newly introduced property, so it is a rare resource to see. However, gap is still a very useful property that is safe to use.

Navigation

The most common way of navigation is to have a list of items displayed along the interior of a horizontal bar. Doing so often leaves us with extra space. This additonal space can be displayed outside of the item, resulting in items being spaced from each other with a whitespace between or around said items.

    Space distributed outside the item. Example.
  • Use alignment properties and justify-content. In the example above, items are displayed at their natural size and with justify-content: space-between, they are also spaced equally. The whitespace size can be controlled using space-around or space-evenly values. start can be used to insert whitespaces at the end of items, or end to insert before the item, or center to center navigation items.

      Summary:
    • Methods of controlling whitespace sizes through the values to justify-content: :
      • space-between distributes whitespace evenly, only between items.
      • space-around distributes all available whitespace around individual items.
      • space-evenly distributes all available whitespace around and between all items equally.
      • Use start to place all available whitespace at the end of items.
      • Use end to place all available whitespace before the items.
      • Use center to place all items in the center, gapless, and distribute all remaining whitespace before and after items.

    • Space distributed inside the item Example.
  • Another method of whitespace distribution is within the items instead of creating whitespace between items. This is done through using the flex properties. Refer back to the lesson on growing and shrinking.

Split Navigation

Another method to align items on the main axis is to use auto margins. This allows different groups of items to be aligned left and others aligned right.

Example

Items are aligned on the main axis to justify-content with the value normal. normal behaves as start (justify-content: flex-start), the initial behavior of flexbox. Using gap creates a literal gap between items. Also, the last item is pushed to the right with the property margin-left with a value of auto. By placing the class selector on the appropriate item, you can see where the split happens.


Summary

    The following values to the property justify-content:
  • normal behaves the same as start (the default to Flexbox).

    • gap creates a literal gap between items with value stated.

Center Item

Example.

With flexbox, you can apply the value center to justify-content and align-items to center items. However, it can be done with CSS box alignment properties as well. First, remove display: flex from the container, as well as the justify-content and align-items properties. Afterwards, to achieve vertical centering, use align-content: center. To finish with horizontal centering, apply margin: auto to the child element.
