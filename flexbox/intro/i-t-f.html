What is Flexbox?

Flexbox is a method of arranging items into rows or columns. They will flex (i.e grow or shrink) based on the rules defined.

Example 1 

                    HTML
                    
                        <div class="flex-container">
                            <div class="one"></div>
                            <div class="two"></div>
                            <div class="three"></div>
                        </div>
                    
                
                    CSS
                    
                        .flex-container {
                            /* display: flex; */
                          }
                          
                          /* this selector selects all divs inside of .flex-container */
                          .flex-container div {
                            background: peachpuff;
                            border: 4px solid brown;
                            height: 100px;
                            /* flex: 1; */
                          }

In the example, when you remove the comments tags in the CSS declarations (/* and */), you set all three <div>s to be arranged horizontally. When hitting the 1x, 0.5x, 0.25x magnifiers, the spacing between all three are equal even when zoomed out. This is how they 'flex'.

If you add another <div> to the HTML, you will get four <div> tags spread evenly.

Flex Containers and Flex Items


Flexbox is composed a plethora of properties that are divided into two categories:

Flex Container
Any element that has display: flex.
Flex Item
An element that lives directly inside of a flex container.


Any element can be both a flex container and a flex item.

Quiz

What makes an element a flex container?

Any element with the property display: flex is a flex container.

What makes an element a flex item?

Any direct child of a flex container (an element with the display: flex property) becomes a flex item. Descendants deeper than one level are not flex items unless their immediate parent is also a flex container.

How do you create a flex item?

A flex item is made when an element is a direct descendant of another element with a display: flex property.
