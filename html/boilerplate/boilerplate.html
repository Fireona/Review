Boilerplate

<!DOCTYPE> tags must be used to indicate HTML version.

<html> tags indicate the start of the html document. Known as the root.

  • "lang=en" attribute indicates language used for html document.

<head> tags are used to provide instructions for the boilerplate.

<meta charset="UTF-8"> sets encoding ensures special symbols and characters from different languages correctly in the browser.

<title> provides the boilerplate context of the webpage with a title.

<body> provides html document with content.

<h1> provides a heading to the document. Ranges from h1 to h6, from biggest to smallest respectively.
