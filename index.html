Boilerplate

<!DOCTYPE> tags must be used to indicate HTML version.

<html> tags indicate the start of the html document. Known as the root.

  • "lang=en" attribute indicates language used for html document.

<head> tags are used to provide instructions for the boilerplate.

<meta charset="UTF-8"> sets encoding ensures special symbols and characters from different languages correctly in the browser.

<title> provides the boilerplate context of the webpage with a title.

<body> provides html document with content.

<h1> provides a heading to the document. Ranges from h1 to h6, from biggest to smallest respectively.

Body elements

Create paragraph tags using <p>.

Use <strong> To indicate words of importance, seriousness, and urgency without changing the meaning of the content.

  • Except in emergencies, do not press the emergency button.

Use <em> in order to emphasize words that change the mean of the content.

  • A person could be tired because he was up all night.

Use <i> to change the mood or voice of words from surrounding content. Often used with foreign words, idiomatic phrases, technical terms, ship names, thoughts, etc.

  • Even when we do have fun, people are become... over rambunctious.

Use <b> to bring attention to words without indicating importance, serious, or urgency.

  • If I have what I wanted, I thought I'd be happy. Now... I want more.

<!-- --> creates hidden comments in the line not visible to users.

In order semantics (standards), from most to least important, goes as <strong>, <em>, <i>, and lastly <b>.

Lists

<ul> creates unordered lists. Used for things like getting groceries, when order is not a neccessity.

Groceries

  • Meat
  • Veggies

<ol> creates ordered. Used for things like directions in a recipe.

Recipe

  1. Preheat oven.
  2. Wash veggies.
  3. Line pan with veggies.
  4. Once oven is preheated, place pan in oven. Bake for 30 minutes.

<li> is used in both ordered and unordered lists to create listed items.

Links and Images

Links

Use <a>, an anchor tag, to create links.

There are:

  • Absolute Links
  • Relative Links

Each anchor tag uses an href attribute that designates the image location.

To go to YouTube, input "https://www.youtube.com/".

YouTube

If we want to open in a new tab, add " target="_blank"" after the href attribute.

YouTube

However, if left as is, this leaves security issues open with JavaScript. For this reason, we add " rel=noopener noreferrer " after the target attribute.

YouTube

"noopener" is a safety protocol. Itensures the newly opened tab that it can not access or interact with the page opened from.

"norefferer" provides the benefits of noopener, but also hides information of where the link came from.

Without using "noopener", this leads to security issues that are called phishing attacks where the opened link has the possibility of changing the original webpage to something different. This is known as tabnapping.

Images

When bringing in images onto a webpage, use <img> tag.

<img> tags are void elements, so a closing tag is not necessary.

However, just like anchor tags, they do have a destination attribute similar to href. For images, src is used.

We must add an alt attribute to describe the image for the visually impaired as well.

Unfortunately the provided image is too large, so we can specify the image size with 'height=' and 'width='.
