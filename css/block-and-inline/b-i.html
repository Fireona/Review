Block vs Inline

Most elements we've used so far are block elements, or another way of saying it is their default style is set to "display: block". By default, all block elements will appear on the page stacked on top of each other. Every new block element will create a new line.

On the contrary, inline elements do not create a new line, but appear in line with the elements they are placed next to. A good example would be a link (from using the <a> tags) that is part of a text content. For example:

Blah blah blah blah blah blah blah blah.


As you can see that even though there is a link inside, the line of text is sitting inside as part of the content.
